Description

Lap Recorder is a Torn racing helper focused on capturing lap times, keeping local records useful, and giving each session enough context to review later without losing the shape of the actual race.

What this script does

Lap capture

Records racing laps with visible last-lap data, calculated chrono, delta, and race context for cleaner review.

Track and car records

Best Lap always uses the base circuit name, while Race Time uses the base track for official layouts and only splits into Trackname - X Laps for custom races.

Replay context

Helps connect recorded times with replay-aware details for easier post-race review, without replay auto-clicking.

Recorder window

Opens at a larger default size, now with a taller default and minimum height, and stays inside the viewport when opened, restored, dragged, resized, or after a browser resize.
