What is EggsTerminator?

EggsTerminator is a Torn Easter event userscript that combines page movement with egg visibility and progress tracking tools.

It is meant to make the event run feel less scattered by keeping navigation and finder context together.

How do I install it?

Install it through a userscript manager such as Tampermonkey, or through Torn PDA if the release build supports it.

Reload Torn after installation, then use the script during the Easter event pages and routes it supports.

What does state-aware navigation mean?

State-aware navigation means the script is built around the current event state instead of blindly moving through pages.

The goal is to help you traverse pages while keeping the egg hunt workflow in view.

Why are found and collected counters separate?

Found and collected can describe different parts of the event workflow.

Separating those counts makes it easier to tell whether you have spotted an egg, collected it, or still need to act on it.

Can I use it outside the Easter event?

The script description is specifically for Torn's Easter event, so its main value is during that event window.

Outside the event, navigation may still load, but the finder and counter workflow may not have anything useful to track.

Where is progress data stored?

As a userscript, any saved counters or preferences should be treated as local userscript-manager or browser data unless the release notes say otherwise.

Clear browser storage carefully if you want to keep event progress between sessions.

The navigator is not moving as expected. What should I check?
  • Make sure the Easter event is active.
  • Reload Torn after installing or updating the script.
  • Confirm the userscript is enabled in Tampermonkey or Torn PDA.
  • Check whether the page you are on is part of the supported navigation path.
How do I report an issue?

Include the page you were on, what the counter showed, what navigation step you expected, and whether an egg was visible or already collected.

That makes event-state issues much easier to trace.
