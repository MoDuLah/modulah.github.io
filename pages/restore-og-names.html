Description

Restore OG Names is a lightweight Torn interface utility for bringing familiar classic-style naming back to the views where the original wording is easier to recognise.

What this script does

Classic-style names

Restores familiar naming where newer labels make Torn pages feel less recognisable.

Lightweight changes

Focuses on text replacement and small page enhancements rather than heavy interface changes.

Clearer page reading

Helps relevant views read closer to the naming style long-time players expect.

Familiarity first

Keeps the tool intentionally simple: restore the wording, keep browsing, and move on.
