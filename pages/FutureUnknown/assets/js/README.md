# FutureUnknown JS structure

- app.js — bootstrap / startup
- core/state.js — central game state
- core/utils.js — shared helpers and formatters
- systems/log.js — system log and timestamps
- systems/player.js — player actions / XP / feed / exercise / study
- systems/passive.js — passive recovery, decay, shield recharge, freight updates
- systems/events.js — travel encounters and interception handling
- systems/salvage.js — debris, salvage, repair, recycling
- systems/travel.js — travel lifecycle and transit timers
- systems/market.js — market listings, purchases, cargo deliveries
- ui/render.js — page rendering and UI flow
