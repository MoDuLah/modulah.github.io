Description

Stock-X is a stock-focused Torn helper that brings vault visibility, ROI context, and faster market-check support into a cleaner investing workflow.

What this script does

Stock workflow helpers

Keeps common stock-checking tasks easier to scan while you review market movement.

Vault visibility

Surfaces vault-focused context so stored value and stock activity are easier to keep in view.

ROI context

Frames investing decisions around return and value instead of only raw stock listings.

Faster market checks

Designed around quick reads, cleaner visibility, and fewer repeated manual checks.
