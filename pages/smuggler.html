Description

Smuggler is the renamed identity for the former Travel Assistant page on the hub. It keeps the same travel-focused direction around destination checks, flight timing, and trip planning, while the dedicated branding and documentation continue to be expanded.

What this script does

Destination context

Keeps destination-focused travel information easier to review before and during a trip.

Flight timing

Supports planning around travel time, return timing, and the next step in your route.

Run planning

Helps keep the purpose of each travel run clear, whether you are checking destinations or preparing the next flight.

Lightweight workflow

Focuses on useful travel context without turning the page into a heavy dashboard.
