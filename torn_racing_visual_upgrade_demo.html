Torn Racing Visual Upgrade Demo

This is a proof-of-concept showing how one base car can visually change through material swaps, overlay layers, and attachment slots. It demonstrates why cosmetic combinations do not necessarily require one complete car model per permutation.
Base car + selected visual layers
Assets used: 1 base + 0 active layers

Upgrade sections

Developer point: this demo does not create a new image/model for every possible combination. It keeps one base car and changes visible layers such as paint material, wheels, spoilers, carbon panels, brake details, exhaust tips, safety equipment, and ride height.
