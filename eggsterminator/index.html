Description

EggsTerminator is a Torn Easter event helper that combines guided page traversal, egg visibility support, and found-or-collected counters in one lightweight workflow.

What this script does

State-aware navigation

Helps move through Torn pages in a way that respects the event state you are currently tracking.

Egg visibility support

Keeps the finder side of the workflow focused on spotting event eggs without extra clutter.

Found and collected counters

Separates what has been found from what has already been collected, making progress easier to follow.

Navigator and finder together

Combines page movement and event tracking so the Easter run feels less scattered.
